Access the MJII Procedures Master File

How to Access Procedures

Option 1: Direct File Access

If you're on the yacht's network with the S: drive mapped:

S:\Engineering\Admin\Procedures Master File

Option 2: Windows Explorer

  1. Open Windows Explorer
  2. Navigate to: S:\Engineering\Admin\Procedures Master File
  3. Browse the procedure files

Option 3: Network Access

If the S: drive isn't mapped:

  1. Open Windows Explorer
  2. Type in the address bar: \\server-name\Engineering\Admin\Procedures Master File
  3. Replace "server-name" with the actual server name

Need Help?

Contact the IT department or engineering team if you need assistance accessing the procedures.
